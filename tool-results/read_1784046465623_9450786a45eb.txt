    32→## ⚡ Quick Start
    33→
    34→### Option 1: Install the app (Recommended)
    35→
    36→The fastest way in — no clone, no terminal. [**Agency Agents**](https://agencyagents.app) is a native desktop app (macOS · Linux · Windows) that browses the whole roster and installs agents into Claude Code, Cursor, Codex, Gemini CLI, OpenCode, Qwen, and Osaurus for you, then keeps them up to date.
    37→
    38→**[⬇ Download the latest release](https://github.com/msitarzewski/agency-agents-app/releases/latest)** — or on a Mac:
    39→
    40→```bash
    41→brew install --cask msitarzewski/agency-agents/agency-agents
    42→```
    43→
    44→Prefer the command line? The script-based options below install the same agents.
    45→
    46→### Option 2: Use with Claude Code
    47→
    48→```bash
    49→# Install all agents to your Claude Code directory
    50→./scripts/install.sh --tool claude-code
    51→
    52→# Or manually copy a category if you only want one division
    53→cp engineering/*.md ~/.claude/agents/
    54→
    55→# Then activate any agent in your Claude Code sessions:
    56→# "Hey Claude, activate Frontend Developer mode and help me build a React component"
    57→```
    58→
    59→### Option 3: Use as Reference
    60→
    61→Each agent file contains:
    62→- Identity & personality traits
    63→- Core mission & workflows
    64→- Technical deliverables with code examples
    65→- Success metrics & communication style
    66→
    67→Browse the agents below and copy/adapt the ones you need!
    68→
    69→### Option 4: Use with Other Tools (GitHub Copilot, Antigravity, Gemini CLI, OpenCode, OpenClaw, Cursor, Aider, Windsurf, Kimi Code, Codex, Osaurus, Hermes, Mistral Vibe)
    70→
    71→```bash
    72→# Step 1 -- generate integration files for all supported tools
    73→./scripts/convert.sh
    74→
    75→# Step 2 -- install interactively (auto-detects what you have installed)
    76→./scripts/install.sh
    77→
    78→# Or target a specific tool directly
    79→./scripts/install.sh --tool antigravity
    80→./scripts/install.sh --tool gemini-cli
    81→./scripts/install.sh --tool opencode
    82→./scripts/install.sh --tool copilot
    83→./scripts/install.sh --tool openclaw
    84→./scripts/install.sh --tool cursor
    85→./scripts/install.sh --tool aider
    86→./scripts/install.sh --tool windsurf
    87→./scripts/install.sh --tool kimi
    88→./scripts/install.sh --tool codex
    89→./scripts/install.sh --tool osaurus
    90→./scripts/install.sh --tool hermes
    91→./scripts/install.sh --tool vibe
    92→```
    93→
    94→**Install only the teams you need** (not everyone wants every division):
    95→
    96→```bash
    97→./scripts/install.sh                                    # interactive wizard: pick tools + teams
    98→./scripts/install.sh --tool claude-code --division engineering,security
    99→./scripts/install.sh --tool cursor --agent frontend-developer,ui-designer
   100→./scripts/install.sh --list teams                       # see every team + agent count
   101→./scripts/install.sh --tool opencode --division engineering --dry-run
   102→```
   103→
   104→> **OpenCode note:** OpenCode's runtime currently registers only ~119 agents and silently drops the rest ([upstream bug](https://github.com/anomalyco/opencode/issues/27988)). Installing a subset with `--division` keeps you under that limit. The installer warns you when a selection would exceed it.

   690→## 🔌 Multi-Tool Integrations
   691→
   692→The Agency works natively with Claude Code, and ships conversion + install scripts so you can use the same agents across every major agentic coding tool.
   693→
   694→### Supported Tools
   695→
   696→- **[Claude Code](https://claude.ai/code)** — native `.md` agents, no conversion needed → `~/.claude/agents/`
   697→- **[GitHub Copilot](https://github.com/copilot)** — native `.md` agents, no conversion needed → `~/.github/agents/` + `~/.copilot/agents/`
   698→- **[Antigravity](https://github.com/google-gemini/antigravity)** — `SKILL.md` per agent → `~/.gemini/config/skills/`
   699→- **[Gemini CLI](https://github.com/google-gemini/gemini-cli)** -- `.md` agent files -> `~/.gemini/agents/`
   700→- **[OpenCode](https://opencode.ai)** — `.md` agent files → `.opencode/agents/`
   701→- **[Cursor](https://cursor.sh)** — `.mdc` rule files → `.cursor/rules/`
   702→- **[Aider](https://aider.chat)** — single `CONVENTIONS.md` → `./CONVENTIONS.md`
   703→- **[Windsurf](https://codeium.com/windsurf)** — single `.windsurfrules` → `./.windsurfrules`
   704→- **[OpenClaw](https://github.com/openclaw/openclaw)** — `SOUL.md` + `AGENTS.md` + `IDENTITY.md` per agent
   705→- **[Qwen Code](https://github.com/QwenLM/qwen-code)** — `.md` SubAgent files → `~/.qwen/agents/`
   706→- **[Kimi Code](https://github.com/MoonshotAI/kimi-cli)** — YAML agent specs → `~/.config/kimi/agents/`
   707→- **[Codex](https://developers.openai.com/codex/overview)** — TOML custom agents → `~/.codex/agents/`
   708→- **Osaurus** -- `SKILL.md` skills -> `~/.osaurus/skills/`
   709→- **[Hermes](integrations/hermes/README.md)** -- lazy-router plugin -> `~/.hermes/plugins/`

   712→
   713→### ⚡ Quick Install
   714→
   715→**Step 1 -- Generate integration files:**
   716→```bash
   717→./scripts/convert.sh
   718→# Faster (parallel, output order may vary): ./scripts/convert.sh --parallel
   719→```
   720→
   721→**Step 2 -- Install (interactive, auto-detects your tools):**
   722→```bash
   723→./scripts/install.sh
   724→# Faster (parallel, output order may vary): ./scripts/install.sh --no-interactive --parallel
   725→```
   726→
   727→The installer scans your system for installed tools, shows a checkbox UI, and lets you pick exactly what to install:
   728→
   729→```
   730→  +------------------------------------------------+
   731→  |   The Agency -- Tool Installer                 |
   732→  +------------------------------------------------+
   733→
   734→  System scan: [*] = detected on this machine
   735→
   736→  [x]  1)  [*]  Claude Code     (claude.ai/code)
   737→  [x]  2)  [*]  Copilot         (~/.github + ~/.copilot)
   738→  [x]  3)  [*]  Antigravity     (~/.gemini/antigravity)
   739→  [ ]  4)  [ ]  Gemini CLI      (~/.gemini/agents)
   740→  [ ]  5)  [ ]  OpenCode        (opencode.ai)
   741→  [ ]  6)  [ ]  OpenClaw        (~/.openclaw/agency-agents)
   742→  [x]  7)  [*]  Cursor          (.cursor/rules)
   743→  [ ]  8)  [ ]  Aider           (CONVENTIONS.md)
   744→  [ ]  9)  [ ]  Windsurf        (.windsurfrules)
   745→  [ ] 10)  [ ]  Qwen Code       (~/.qwen/agents)
   746→  [ ] 11)  [ ]  Kimi Code       (~/.config/kimi/agents)
   747→  [ ] 12)  [ ]  Codex           (~/.codex/agents)
   748→  [ ] 13)  [ ]  Osaurus         (~/.osaurus/skills)
   749→  [ ] 14)  [ ]  Hermes          (~/.hermes/plugins)
   750→
   751→  [1-14] toggle   [a] all   [n] none   [d] detected
   752→  [Enter] install   [q] quit
   753→```
   754→
   755→**Or install a specific tool directly:**
   756→```bash
   757→./scripts/install.sh --tool cursor
   758→./scripts/install.sh --tool opencode
   759→./scripts/install.sh --tool openclaw
   760→./scripts/install.sh --tool antigravity
   761→./scripts/install.sh --tool codex
   762→./scripts/install.sh --tool osaurus
   763→./scripts/install.sh --tool hermes
   764→```
   765→
   766→**Non-interactive (CI/scripts):**
   767→```bash
   768→./scripts/install.sh --no-interactive --tool all
   769→```
   770→
   771→**Faster runs (parallel)** — On multi-core machines, use `--parallel` so each tool is processed in parallel. Output order across tools is non-deterministic. Works with both interactive and non-interactive install: e.g. `./scripts/install.sh --interactive --parallel` (pick tools, then install in parallel) or `./scripts/install.sh --no-interactive --parallel`. Job count defaults to `nproc` (Linux), `sysctl -n hw.ncpu` (macOS), or 4; override with `--jobs N`.
   772→
   773→```bash
   774→./scripts/convert.sh --parallel                    # convert all tools in parallel
   775→./scripts/convert.sh --parallel --jobs 8           # cap parallel jobs
   776→./scripts/install.sh --no-interactive --parallel   # install all detected tools in parallel
   777→./scripts/install.sh --interactive --parallel      # pick tools, then install in parallel
   778→./scripts/install.sh --no-interactive --parallel --jobs 4
   779→```
